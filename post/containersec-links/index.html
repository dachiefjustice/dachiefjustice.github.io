Container Security Part V

Links & Resources

 Posted on April 2, 2018  |  2 minutes (239 words)

This is the fifth post in a series on container security. Check out the first post for context and links to the rest of the series.

Want to go deeper on container security? Check out the links below.

Docker Security Links

  • Docker’s security documentation
  • Docker’s security best practices focusing on Docker Enterprise Edition
  • Docker’s reference architectures are a collection of reference architectures/design patterns, demonstrating best practices for a variety of use cases
  • Docker seccomp lab
  • GDS Security’s Docker Secure Deployment Guidelines, a checklist for securing Docker deployments
  • Docker Bench for Security is a script (and container) that checks Docker deployments against a variety of security best practices

Kubernetes Security Links

K8s is a popular container orchestration platform, here are some handy links for securing it.

  • Kubernetes CIS benchmark
  • Kubernetes CIS benchmark InSpec profile
  • Kubernetes Bench for security
  • Kubernetes security best practices blog post

Other Links

  • NCC Group whitepaper on Understanding and Hardening Linux Containers provides an in-depth look at the Linux container ecosystem and explores hardening options
  • NCC Group whitepaper on Abusing Privileged and Unprivileged Linux Containers provides a penetration tester/offense perspective on Linux-based containers
  • Dan Walsh’s post on securing containers with kernel capabilities
  • RedHat’s Container Security Guide
  • Jess Frazelle’s blog and contained.af (a container security game)
  • Info on running non-privileged containers, a collection of resources on running containers without excessive privileges
  • NIST’s Application Container Security Guide SP 800-190 provides an architecture-level, technology-agnostic look at container risks and mitigations
