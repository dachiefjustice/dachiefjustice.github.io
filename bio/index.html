Bio

Application & AWS Security

Hi there! My name’s John Roberts, you can find me on GitHub or LinkedIn, or drop me an email.

I help secure AWS environments and web applications. I’ve worked as a security consultant with Cigital (a software security firm acquired by Synopsys). Some highlights of my work:

  • Led, scoped, and delivered web app/API penetration testing, network penetration testing, design/architecture analysis, and red teaming projects
  • Designed & delivered a service to analyze the security of client AWS environments, built it into a repeatable service offering, and trained other consultants on delivery
  • Led creation and delivery of a training workshop for DevOps and security teams (focused on AWS, Docker, and infrastructure-as-code security)

Before Cigital, I worked as a network and systems administrator at a startup. I managed applications & infrastructure on AWS, and built out the corporate network from scratch.

I have a B.S. from Rochester Institute of Technology, graduating summa cum laude with a 4.0 in-major GPA (Information Technology, Networking and Information Security concentrations).

If I’m not helping clients or playing with tech, I’m probably out hiking! In 2017, I backpacked ~2,000 miles of the Pacific Crest Trail.
